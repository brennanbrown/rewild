# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-12-03

- Added `SITE-SPEC.md` describing the 11ty site design.
- Created `TODO.md` to track project tasks.
- Initialized a minimal Eleventy project structure under `src/`.

## [0.2.0] - 2025-12-03

- Configured Eleventy collections for framework, practices, and workshop content.
- Updated layouts to use `layouts/base.njk` and ensured pages share a common base template.
- Expanded the Framework overview page with summaries of the five strands.
- Added initial practice pages: "Freewriting as Rewilding" and "Succession Writing".
- Expanded the Wild Minds Workshop overview with core elements and a sample arc.
- Expanded the Access and Ecologies page with access commitments and adaptation guidance.
- Populated the Resources page with a core reading list based on the essay.

## [0.3.0] - 2025-12-03

- Integrated Tailwind CSS with a custom color palette and typography for the site.
- Added the Tailwind Typography plugin for nicer default styling of long-form content.
- Introduced a new layout structure using Tailwind utility classes.
- Linked Font Awesome via CDN and added icons to the main navigation and header.
